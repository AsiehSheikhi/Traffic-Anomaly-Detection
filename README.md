Traffic Anomaly Detection
Detecting anomalies in traffic data using Machine Learning (Python Project)

این پروژه شامل یک تحلیل مقدماتی داده‌های ترافیکی و پیاده‌سازی ساده روش‌های تشخیص ناهنجاری (Anomaly Detection) است.
هدف این پروژه، شروع مسیر یادگیری در حوزه هوش مصنوعی در حمل‌ونقل است.

 ویژگی‌های پروژه

 پردازش و پاک‌سازی داده‌ها
 تحلیل آماری اولیه (EDA)
 رسم نمودارهای توزیع، همبستگی و الگوهای داده
 پیاده‌سازی مدل‌های ساده تشخیص ناهنجاری
 استفاده از کتابخانه‌های استاندارد پایتون:
NumPy • Pandas • Matplotlib • Seaborn • Scikit-Learn
